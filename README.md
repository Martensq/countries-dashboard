# Countries Dashboard

A lightweight React dashboard to explore country data using the REST Countries API.

![Cards](./public/cards.png)

## Tech stack
- React + Vite
- Tailwind CSS
- React Router (HashRouter for GitHub Pages)

## Features
- Fetches countries from REST Countries
- Search by name
- Filter by region
- Sorting (name / population)
- Pagination
- Toggle view: Cards / Table
- Country details page (flag, region, capital, population, languages, currencies, borders)

## Getting started
`npm install`

`npm run dev`

## Build
`npm run build`

`npm run preview`

## Deploy (GitHub Pages)
`npm run deploy`

## Data source
REST Countries API: https://restcountries.com/
